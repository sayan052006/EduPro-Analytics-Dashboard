\# 📊 Instructor Performance and Course Quality Evaluation on EduPro



\## 📌 Project Overview



This project analyzes instructor performance and course quality on the EduPro online learning platform. The dashboard helps identify top-performing instructors, evaluate course quality, and understand the relationship between teaching experience and learner satisfaction.



The project is built using \*\*Python\*\*, \*\*Pandas\*\*, \*\*Plotly\*\*, and \*\*Streamlit\*\*.



\---



\## 🎯 Objectives



\- Analyze instructor performance.

\- Compare teacher ratings and course ratings.

\- Identify top-performing instructors.

\- Analyze course quality by category and level.

\- Study the impact of teaching experience on ratings.

\- Build an interactive dashboard for data visualization.



\---



\## 📂 Dataset



The project uses an Excel dataset containing four sheets:

\## 🚀 How to Run the Project



\### 1. Install the required libraries



```bash

pip install -r requirements.txt

```



\### 2. Open the project folder



```bash

cd EduPro\_Project

```



\### 3. Run the Streamlit application



```bash

streamlit run app.py

```



\### 4. Open the browser



```

http://localhost:8501

```

\## 📊 Key Insights



\- Highly rated instructors generally receive better course ratings.

\- Experienced instructors tend to perform better in learner ratings.

\- Machine Learning and Cybersecurity instructors show strong performance.

\- Course quality varies across different categories.

\- The dashboard helps identify top-performing instructors and improvement areas.
